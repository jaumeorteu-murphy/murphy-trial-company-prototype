The clever bit: importing policies across companies

A policy points at a product category by an ID that’s unique per company, so we can’t paste it as-is. We match on the category’s stable code instead — so the right category always lines up.

Reference pack
Credit cards id src-CC
Loans id src-LO
match on code
credit_cards loans
New trial company
Credit cards id new-CC
Loans id new-LO

Every policy filter is rewritten src-CC → new-CC, src-LO → new-LO. Join on the code, never the ID or the name. If a code can’t be matched, we stop and flag it — never drop a policy silently.
